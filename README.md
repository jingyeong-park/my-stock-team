# my-stock-team 📈

**종목 이름만 말하면, AI 애널리스트 팀이 알아서 조사하고 발표자료(PPT·PDF)까지 만들어 주는 도구입니다.**

예를 들어 `삼성전자`라고 입력하면 —
재무·주가·뉴스·리스크를 각각 담당하는 AI들이 나눠서 조사한 뒤, 한 편의 리포트로 정리해 줍니다.

> ⚠️ **먼저 알아두세요:** 이 도구는 **공개된 무료 데이터로 만든 학습용**입니다.
> "이 주식 사라/팔아라" 같은 **투자 권유는 하지 않으며**, 최종 판단은 본인 몫입니다.

---

## 🧑‍🤝‍🧑 누가 무슨 일을 하나요? (AI 팀원 5명)

| 팀원 | 하는 일 | 쉽게 말하면 |
|---|---|---|
| 재무 애널리스트 | 회사의 매출·이익·공시 분석 | "이 회사 돈은 잘 버나?" |
| 차트 애널리스트 | 주가·거래량·추세 분석 | "주가 흐름은 어땠나?" |
| 뉴스 애널리스트 | 최근 뉴스·시장 분위기 분석 | "요즘 분위기는 어때?" |
| 리스크 매니저 | 위험 요소 정리 | "조심할 점은 뭐지?" |
| 검증 애널리스트 | 리포트에 틀린 게 없는지 점검 | "이거 내보내도 되나?" |

이 5명이 협업해서 만든 결과를 **PPT/PDF 리포트**로 받아볼 수 있습니다.

---

## 🛠️ 시작하기 전에 준비물 3가지

처음이라면 아래 3개를 차례로 준비하세요. **한 번만 설치하면 됩니다.**

### 1) Claude Code (이 도구를 실행하는 프로그램)
이미 이 글을 Claude Code 안에서 보고 있다면 ✅ 준비 끝입니다.
처음이라면 [claude.com/claude-code](https://claude.com/claude-code) 에서 설치하세요.

### 2) 파이썬(Python) — 리포트를 그려주는 엔진
컴퓨터에 파이썬이 있어야 PPT/PDF가 만들어집니다.
- 설치: [python.org/downloads](https://www.python.org/downloads/) 에서 내려받아 설치
- 설치할 때 **"Add Python to PATH"** 체크박스를 꼭 켜세요.
- 설치 후, 아래 한 줄을 복사해 실행하면 필요한 부품들이 한 번에 깔립니다:
  ```
  pip install python-pptx fpdf2 finance-datareader pykrx opendartreader pandas
  ```
  > 💡 위 명령은 Claude Code 입력창에 `!`를 붙여 `! pip install ...` 형태로 실행하거나,
  > 윈도우 검색창에 "PowerShell"을 열어 붙여넣어도 됩니다.

### 3) DART 열쇠(API 키) — 재무 데이터를 가져오는 무료 열쇠
금융감독원이 운영하는 무료 공시 사이트의 출입증 같은 것입니다. **무료이고 1분이면 발급됩니다.**
1. [opendart.fss.or.kr](https://opendart.fss.or.kr) 접속 → "인증키 신청/관리" → 이메일로 신청
2. 받은 키(영어+숫자 긴 문자열)를, 작업하는 폴더의 `.env` 파일에 아래처럼 적어두세요:
   ```
   DART_KEY=여기에_발급받은_키를_붙여넣기
   ```
   > 🔒 이 키는 **나만 아는 비밀번호**처럼 다루세요. 남에게 보여주거나 인터넷에 올리지 마세요.
   > (이 도구에는 어떤 키도 들어있지 않습니다. 각자 본인 키를 넣어야 합니다.)

   재무 분석이 필요 없다면 키 없이도 차트·뉴스 분석은 됩니다.

## 📥 설치 방법 (복사 → 붙여넣기 2줄)

Claude Code 입력창에 아래 두 줄을 차례로 입력하세요. (앞의 `/`까지 그대로)

```
/plugin marketplace add jingyeong-park/my-stock-team
```
```
/plugin install my-stock-team@my-stock-team-marketplace
```

- 첫 줄: "이런 도구 모음이 있어요"라고 Claude에게 알려주는 단계
- 둘째 줄: 그중 `my-stock-team`을 실제로 설치하는 단계

설치가 끝나면 준비 완료입니다. 🎉

## ▶️ 사용 방법 (이게 전부예요)

Claude Code 입력창에 이렇게 입력하세요:

```
/stock-research 삼성전자
```

여러 종목을 한 번에 보고 싶다면:

```
/stock-research 삼성전자 SK하이닉스
```

그러면 AI 팀이 알아서 조사 → 정리 → 리포트 초안까지 만들어 줍니다.
중간에 **"PDF로 만들어줘"** 라고 하면 발표용 PDF 파일까지 뽑아줍니다.

> 굳이 `/stock-research`를 안 쳐도, 그냥 **"삼성전자 주가 흐름 어때?"** 처럼 말로 물어봐도 됩니다.

### 결과물은 어디에 생기나요?
작업 폴더 안 `reports/` 폴더에 만들어집니다.
- `reports/삼성전자.md` — 글로 정리된 리포트
- `reports/삼성전자.pptx` — 파워포인트 발표자료
- `reports/삼성전자.pdf` — PDF 발표자료

## ❓ 잘 안 될 때 (자주 묻는 것)

| 이런 일이 생기면 | 이렇게 해보세요 |
|---|---|
| "pip을 찾을 수 없음" 같은 메시지 | 파이썬이 안 깔렸거나 PATH 체크를 안 한 경우입니다. 2번 준비물을 다시 확인하세요. |
| 재무 수치가 "확인 불가"로 나옴 | DART 키(3번)가 없거나 잘못 적힌 경우입니다. `.env` 파일을 확인하세요. |
| 글자가 깨져 보이는 PPT/PDF | 리포트는 "맑은 고딕" 글꼴을 씁니다. 윈도우에는 기본 탑재돼 있습니다. |
| 그래도 안 되면 | Claude Code에게 그냥 **"리포트 만들다 이런 오류가 났어"** 라고 화면을 보여주며 물어보세요. |

## 🔒 안전·신뢰 약속 (모든 리포트에 항상 적용)

- 모든 숫자에는 **출처와 날짜**를 함께 적습니다. 출처 없는 숫자는 쓰지 않습니다.
- 못 구한 정보는 지어내지 않고 **"확인 불가"**, 소문은 **"미확인"**으로 표시합니다.
- **사라/팔아라/목표가 같은 단정은 하지 않습니다.** 판단에 필요한 근거만 정리합니다.
- 리포트 처음과 끝에 "학습용"이라는 안내와 사용한 데이터 출처를 답니다.

> 본 도구의 산출물은 무료 공개 데이터 기반 **학습용**이며, **투자 권유가 아닙니다.** 투자의 최종 책임은 본인에게 있습니다.
